Sparse symmetric nonnegative matrix factorization applied to face recognition

H. Dobrovolskyi, N. Keberle and Y. Ternovyy, "Sparse symmetric nonnegative matrix factorization applied to face recognition," 2017 9th IEEE International Conference on Intelligent Data Acquisition and Advanced Computing Systems: Technology and Applications (IDAACS), Bucharest, Romania, 2017, pp. 1042-1045. doi: 10.1109/IDAACS.2017.8095244

International Conference on Intelligent Data Acquisition and Advanced Computing Systems: Technology and Applications (IDAACS), Bucharest, Romania, 2017

The task of Sparse Symmetric Nonnegative Matrix Factorization(SSNMF) is formulated as optimization problem and solved numerically with the method of projected gradients descent. The adjustable sparsity level allows to emphasize the most significant object features. Clustering of the Yale Faces data set shows that SSNMF provides the same level of quality as common clustering approaches.

Keywords: Face, Symmetric matrices, Sparse matrices, Face recognition, Feature extraction, Conferences, Optimization, Face Recognition, Symmetric Nonnegative Matrix Factorization, Sparsity Condition
